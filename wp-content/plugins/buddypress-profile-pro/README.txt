== Description ==

This plugin provides an interface to create extended fields and repeater groups for BuddyPress.This plugin provides Extended Fields menu at BuddyPress profile to fill extended fields detail.Admin setting is provided to update fields, group information and to create new profile fields and groups.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Download the zip file and extract it.
2. Upload `buddypress-profile-pro` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the \'Plugins\' menu.
4. Alternatively, you can use WordPress Plugin installer from Dashboard->Plugins->Add New to add this plugin
5. Enjoy

== Frequently Asked Questions ==

= Does This plugin requires BuddyPress? =
Yes, It needs you to have BuddyPress installed and activated.

= Where I can see and create extended fields? =
You can create or see saved extended fields at BuddyPress profile under Profile menu tab.

= How to keep only desired fields in the form? =
You can set fields in the form by selecting desired fields in admin settings under Field Settings tab.

= What is the use of API Key option provided in general settings section? =
Google Places API key is used for setting fields to fill places with google autocomplete.

= How can I create a new extended field form? =
You can create a new field in the form with the help of form provided under Field Settings tab.

= What is the use of Repeater setting provided for fields under Field Settings section? =
Fields which are saved with Repeater option checked provides an option to users to fill data for that particular field multiple times.
